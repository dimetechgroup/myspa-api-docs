# Myspa ERP System — API Documentation

Public developer documentation for integrating with the Spa Management SaaS API.

## Available APIs

- [Lead Capture API](docs/lead-capture-api.md) — push leads into the CRM from external sources (website forms, landing pages, funnels).

## About

This repository hosts the public-facing API reference for developers and integration partners. Documentation is written in Markdown and renders directly on GitHub, or can be published as a website via GitHub Pages (see below).

## Support

Questions about API access or your API key? Contact your account admin or support at [support@myspa.co.ke](mailto:support@myspa.co.ke).
